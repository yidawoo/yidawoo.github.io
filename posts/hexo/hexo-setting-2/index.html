HEXO 添加评论功能

本文记录一下添加评论功能的过程，选择的是 Giscus。

1. 创建仓库

新建一个公开的 GitHub 仓库，用于存放评论，这里设为 blog-comments。在仓库设置中的 Feature 部分勾选开启 Discussions。

2. 安装 Giscus

进入 Giscus ,点击 Install，选择 Only select repositories，
选择刚创建的评论仓库。

访问 Giscus ，填写创建的 用户名/仓库名, Discussion 分类选择推荐的 公告(announcements) 类型即可。

在 启用 giscus 部分，我们将看到 Giscus 的配置

3. 配置 HEXO

编辑 HEXO 中的 _config.butterfly,yml 文件，找到 comments 部分。按照上节配置填写。

Done!
