一、前言

HEXO 是一个快速、简洁且高效的静态博客框架，
基于 Node.js 运行，可以将我们撰写的 Markdown 文档解析渲染成静态的 HTML 网页。
本文整理了 HEXO 的配置以及美化相关过程，用于搭建静态博客网站。

六、结语

至此，HEXO 基本配置完成，需要注意的是，HEXO 依赖于本地环境，必须在本地完成文章后在部署到 GitHub 上。
但实际使用下来，并没有感觉多麻烦，写完后只需要生成部署一键三连即可，反而本地编辑器使用比较习惯更加方便。
后续就是博客页面的美化、域名等等有机会再更。

为什么开始写博客
写博客的初衷，是为了让自己平静下来。 回望这几年的历程，惊觉自己太过急躁。一直以来都只顾闷头赶路，眼里永远是下一件要完成的事情， “高效”是我不自觉的工作信条。我总是焦虑于时间不够用，事情又太多，因此无论什么事情都只想尽快完成然后做下一件。 尽管确实顺利完成了很多事情，但总有一种虚无感，总觉得自己什么也没做，耗费了许多光阴。虽然事实上确确实实也做了一些工作，也投入了很大心力，但这种虚无感始终挥之不去。 现在想来，其实一直都没有真正扎扎实实地去深入做一件事情。 所以我决定尝试开始写作，希望能够慢下来、静下来。 期待有一天回过头来，打开我的博客，能够看到一路以来的痕迹，能够真切感受到光阴并没有虚度。 也许那时候，脚踏实地感能让我获得真正的平静吧。
